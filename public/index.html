Register

To register your own device send a POST request:
curl -H "Content-Type: application/json" -X POST -d '{"name":"Testdevice","address":"192.168.100.151"}' https://nupnp.com/api/register

Any entry gets deleted after 24h and can only be accessed by the IP address which registered it.
