Register

Any entry gets deleted after 24h and can only be accessed by the same internet connection which registered it.

To register your a device:

curl -H "Content-Type: application/json" -X POST -d '{"name":"Devicename","address":"192.168.1.XX"}' https://nupnp.com/api/register


How to setup on a Raspberry Pi

How to setup on Raspberry Pi

  • 1. Edit the crontab file with crontab -e.
  • 2. To register after a reboot and daily at midnight:
  • @reboot sleep 10 && /usr/bin/curl -H "Content-Type: application/json" -X POST -d "{\"name\":\"$(hostname)\",\"address\":\"$(hostname -I)\"}" https://nupnp.com/api/register
@daily /usr/bin/curl -H "Content-Type: application/json" -X POST -d "{\"name\":\"$(hostname)\",\"address\":\"$(hostname -I)\"}" https://nupnp.com/api/register
  • 4. Save and restart device.
  • 5. Refresh nupnp.com.
